# leetcode 问题分类

这一节, 分别基于问题所属的标签, 和问题编号, 列出问题, 方便索引.

目前采用的刷题顺序:

1. 数学
2. 比特位操作
3. 双指针
4. 数组
5. 链表
6. 哈稀表
7. 字符串
8. 栈
9. 队列
10. 树
11. 回溯
12. 贪心
13. 动态规划
14. 图

## 困难程度

| 难度 | 知识点                            |
|----|--------------------------------|
| 入门 | 数组, 字符串, 链表, 排序                |
| 简单 | 栈, 队列, 哈希表, 双指针                |
| 中等 | 二叉树, 堆, 单调栈, 滑动窗口, 二分, 位运算     |
| 困难 | DP, DFS, BFS, 回溯, 贪心, 并查集, 前缀树 |
